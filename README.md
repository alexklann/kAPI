# kAPI

A fast and modern API client made with Tauri and SolidJS.

## Features
- A modern technology stack for performant development.
<br>More is coming soon :)<br>

## Running
**Prerequisites:**
- [Rust](https://rust-lang.org/)
- [pnpm](https://pnpm.io/)

```bash
# Clone the project
git clone https://codeberg.org/klann/kAPI
cd kAPI

# Install pnpm packages
pnpm install

# Start the app
pnpm tauri dev
```
